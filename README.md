# Animated Eyes

An interactive animated face with eyes that follow your cursor and respond to keyboard shortcuts.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the animated eyes.

## Keyboard Shortcuts

- **H** - Toggle help menu
- **Y** - Smile
- **N** - Frown
- **S** - Surprised
- **X** - Suspicious
- **J** - Jump
- **W** - Wink
- **T** - Head Tilt
- **Esc** - Close help menu

The eyes will follow your mouse cursor around the screen!
